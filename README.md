# Weather
Приложение погоды с возможностью поиска городов по всему миру и просто приятное для использования. Выполнено полностью без использования сторонних библиотек.

# Стек
  - UIKit
  - AutoLayout
  - URLSession
  - UserDefaults
  - SwiftLint
